How embedded arrays will affect update operations:

When assigning an employee to a shift:
  Use $push on the shift document:
       db.shifts.updateOne({ _id: shiftObjectId }, { $push: { employees: empObjectId } })

When removing an employee from a shift:
 Use $pull on the shift document:
       db.shifts.updateOne({ _id: shiftObjectId }, { $pull: { employees: empObjectId } })

Querying shifts for an employee is now a single collection scan:
  db.shifts.find({ employees: empObjectId })
